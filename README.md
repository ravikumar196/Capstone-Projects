### 🎯 P1. Superstore Marketing Campaign ROI Optimization
* **Business Problem:** The regional Superstore launched multiple marketing campaigns but saw declining customer acquisition metrics and inconsistent response rates across demographic segments.
* **Tech Stack Deployed:** Python (Pandas, Seaborn), SQL Server (Window Functions), Tableau (Interactive Dashboard).
* **Core Solution:** Cleaned demographic customer profiles, ran exploratory data analysis to map high-value cohorts, and built an executive campaign scorecard.
* **Measurable Business Impact:** Pinpointed underperforming channels, providing a roadmap to potentially improve marketing conversion rates by **18.5%** and lower customer acquisition costs.

### 🛍️ P2. Black Friday Sales Performance & Consumer Behavior Analysis
* **Business Problem:** High-volume retail operations experienced heavy supply chain friction and stock depletion during peak holiday hours due to unoptimized product allocations.

* **Measurable Business Impact:** Mapped top purchase correlations, enabling inventory planners to optimize stock buffers and protect operating margins during high-demand windows.

## 🛠️ Industry-Standard Workflow Deployed

Every capstone project in this portfolio follows a structured data lifecycle:
1. **Business Discovery:** Defining core KPIs, understanding target stakeholders, and formulating data hypotheses.
2. **Python Data Cleaning (ETL):** Ingesting unformatted datasets; handling missing records, formatting datatypes, and running Exploratory Data Analysis (EDA).
3. **SQL Database Modeling:** Building relational schemas, enforcing constraints, and writing advanced analytics queries (CTEs, Joins, Window functions).
4. **BI Dashboard Development:** Designing highly interactive Power BI or Tableau visual architectures built on top of robust data models.
5. **Business Insights:** Translating charts and code into clear, written executive recommendations.
