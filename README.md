## 📌 **About This Repository**

> I'm a data analyst who turns messy, real-world data into business decisions. This repository houses end-to-end case studies across marketing, retail, finance, insurance, and digital payments — each one built on the same disciplined lifecycle: **Business Discovery → Data Cleaning (ETL) → SQL Modeling → Statistical Analysis → BI Dashboard → Business Insights.**
>
> I work across the full analytics stack — from querying and cleaning raw data in **SQL** and **Python (Pandas, NumPy)**, to validating hypotheses with **statistics and A/B testing**, to building modular, scalable data models in **Power BI (DAX)** and **Excel**, to visualizing patterns in **Matplotlib** and **Tableau**.

## 🛠️ **Industry-Standard Workflow Deployed**

<details>
<summary><strong>🔎 Step 1 — Business Discovery</strong></summary>
<br/>
Defining core KPIs, identifying stakeholders, and forming hypotheses before touching a single dataset.
</details>

<details>
<summary><strong>🧹 Step 2 — Data Cleaning (ETL)</strong></summary>
<br/>
Handling missing records, fixing data types, and performing Python EDA with Pandas and NumPy.
</details>

<details>
<summary><strong>🗄️ Step 3 — SQL Database Modeling</strong></summary>
<br/>
Building relational schemas and writing advanced queries — CTEs, Joins, Window Functions.
</details>

<details>
<summary><strong>📈 Step 4 — Statistical Validation</strong></summary>
<br/>
Hypothesis testing, A/B testing, and significance checks to confirm findings hold up.
</details>

<details>
<summary><strong>📊 Step 5 — BI Dashboard Development</strong></summary>
<br/>
Designing modularized, interactive visual architectures in Power BI or Tableau.
</details>

<details>
<summary><strong>💡 Step 6 — Business Insights</strong></summary>
<br/>
Translating charts and code into clear, actionable executive recommendations.
</details>
